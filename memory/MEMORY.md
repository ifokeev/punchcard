# Memory index

One line per note: `- [Title](note-file.md) — hook`. Notes are markdown with
frontmatter (`name`, `description`, `created`, `updated`, optional `branch`,
optional `superseded_by`) and `[[wikilinks]]`. The graph is the links — there is no
database. Recency ranks on conflict; age is not staleness. Validate a note against
the code before relying on it; supersede or delete when wrong.
